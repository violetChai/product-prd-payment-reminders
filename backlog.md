# Product Backlog – Smart Payment Reminders

## Epic: Payment Reminder System

### Story 1
User Story:
As a customer  
I want to see upcoming payments  
So that I can avoid missing them.

Acceptance Criteria:
- Display upcoming payments
- Show payment name and due date

Priority: High

---

### Story 2

User Story:
As a customer  
I want to receive reminders before payments  
So that I remember to complete them.

Acceptance Criteria:
- Reminder sent 2 days before payment
- Notification appears in app

Priority: High

---

### Story 3

User Story:
As a customer  
I want to add recurring payments  
So that I can track upcoming obligations.

Acceptance Criteria:
- User can add payment name
- User can select date
- Payment appears in upcoming list

Priority: Medium

---

### Story 4

User Story:
As a customer  
I want to disable reminders  
So that I can control notifications.

Priority: Low
