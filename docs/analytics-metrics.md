# Product Analytics Metrics

## Primary Metrics

Daily Active Users (DAU)

Number of users accessing the payments dashboard daily.

Payment Completion Rate

Percentage of users who complete payments after receiving reminders.

Reminder Engagement

Percentage of reminder notifications opened.

---

## Secondary Metrics

Feature Adoption

Number of users enabling reminders.

Missed Payment Rate

Percentage of payments missed before and after feature release.

Customer Satisfaction

User feedback rating for payment reminders.

---
