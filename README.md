# Product Requirements Document + Prototype

This repository contains an example Product Requirements Document (PRD) and a simple working prototype for a digital banking feature.

Feature: Smart Payment Reminders

The goal of the feature is to help users avoid missed or late payments by proactively identifying recurring payment patterns and sending reminders before due dates.

This project demonstrates:

• Product problem definition  
• User story development  
• Feature prioritisation  
• Basic UI prototype implementation  

Technologies used in the prototype:

- JavaScript
- HTML
- CSS

This project was built as part of maintaining hands-on technical understanding while working in product management roles.
